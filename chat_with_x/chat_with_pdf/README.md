# 📄 Chat with PDF (Local)

Load any PDF and chat with it — contracts, reports, textbooks. Works offline. No API key.

This template uses the same engine as [Local PDF Chat](../../starter_apps/local_pdf_chat/). Navigate there for the full implementation.

## Quick Start

```bash
git clone https://github.com/shreelaxmi-11/awesome-on-device-ai.git
cd awesome-on-device-ai/starter_apps/local_pdf_chat
pip install -r requirements.txt
python chat.py --pdf your_document.pdf
```
