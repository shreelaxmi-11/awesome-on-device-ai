# 💻 Chat with Code (Local)

Index a GitHub repo or local folder and ask questions about the codebase. No API key.

This template uses the same engine as [Local Code RAG](../../local_rag/local_code_rag/). Navigate there for the full implementation.

## Quick Start

```bash
git clone https://github.com/shreelaxmi-11/awesome-on-device-ai.git
cd awesome-on-device-ai/local_rag/local_code_rag
pip install -r requirements.txt
python chat.py --repo /path/to/your/project
```
